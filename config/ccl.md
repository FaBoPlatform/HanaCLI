# CCLの設定

## Commandのバージョン

```bash
$ export http_proxy=http://proxy:8080
$ export https_proxy=https://proxy:8080
$ export no_proxy="localhost"
```

```bash
$ export http_proxy=http://user:password@proxy:8080
$ export https_proxy=https://user:password@proxy:8080
```
