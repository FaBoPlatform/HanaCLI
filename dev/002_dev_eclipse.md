# 002 開発環境(CCC)

## 必要なSDK

* [Eclipse(neon) IDE for Java EE Developers](http://www.eclipse.org/downloads/packages/)
* [SAP Development Tools for Eclipse](https://tools.hana.ondemand.com/neon/)

## Eclipseインストール　

Eclipse(neon)をダウンロードし解凍。
/Applicationにコピー。

## Pluginの起動
